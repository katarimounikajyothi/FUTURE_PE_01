# GlowUp Warangal Beauty Salon Website

This project is a premium AI-generated salon website created for the Future Interns Prompt Engineering Internship. The website combines *luxury salon aesthetics* with *modern UI design* and *high-conversion website copy*.

# 🛠️ Core Features Developed
* Premium Hero Section  
* Signature Hair & Beauty Services Showcase  
* Makeover & Style Gallery  
* Custom Appointment Booking Section  
* Cozy Salon Corner  
* Customer Testimonials  
* Booking & Contact CTA  
* Responsive Modern UI Design  
* Glassmorphism & Soft Shadow Cards  
* Instagram-style Salon Presentation  

# 🎨 Design Style
* Modern Luxury Salon  
* Soft Pastel Theme  
* Warm Cozy Lighting  
* Elegant Typography  
* Smooth Hover Effects  
* Mobile-Friendly Layout  

# 🎨 Color Palette
* Rose Pink  
* Cream White  
* Lavender Gray  
* Champagne Gold  
* Soft Peach  
* Silver Accents  

# 🧠 Developed Prompt Logic Framework
* Structured prompts for generating homepage, services, and CTA content  
* Iterative refinement of outputs for clarity and conversion  
* AI-assisted design choices for layout, typography, and color palette  
* Documentation of prompt engineering process in `prompt/`  

# 📂 Repository Structure
* prompt/ → Original prompts used  
* outputs/ → Generated text outputs (homepage, services, CTA)  
* screenshots/ → Screenshots of the published site  
* README.md → Project documentation
